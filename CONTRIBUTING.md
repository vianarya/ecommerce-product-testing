# Contributing to Introduction to Git and GitHub

Terima kasih telah tertarik untuk berkontribusi! Untuk menjaga kualitas proyek ini, harap ikuti panduan berikut:

## Cara Berkontribusi
1. **Fork** repositori ini.
2. Buat **Branch** baru untuk fitur atau perbaikan bug Anda (`git checkout -b fitur-baru`).
3. Lakukan **Commit** pada perubahan Anda dengan pesan yang jelas.
4. **Push** ke branch tersebut (`git push origin fitur-baru`).
5. Buat **Pull Request** ke cabang utama.

## Melaporkan Masalah
Jika Anda menemukan bug atau memiliki saran, silakan buka **Issue** baru di tab Issues pada GitHub.

## Kode Etik
Harap patuhi panduan yang ada di dalam file `CODE_OF_CONDUCT.md` dalam semua interaksi Anda.
